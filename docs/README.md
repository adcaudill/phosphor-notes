# Just the Docs Site

This is a Jekyll site using the [Just the Docs](https://just-the-docs.github.io/just-the-docs/) theme. It serves as the documentation for Phosphor Notes.

## To run locally:

1. Install Ruby and Bundler if you don't have them already.
2. Navigate to the `docs` directory in your terminal.
3. Run `bundle install` to install dependencies.
4. Run `bundle exec jekyll serve` to start the local server.
5. Open `http://localhost:4000` in your browser to view the site.
